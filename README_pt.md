⭐ Projetos de Destaque
📈 Previsão de Séries Temporais (Time Series Forecasting)

Previsão de consumo de energia elétrica no Brasil utilizando modelos estatísticos e de Machine Learning. O projeto cobre desde a engenharia de features (Feature Engineering) até pipelines completas de validação, comparação de modelos e forecast.

🔗 Repositório: Time Series Forecasting
🧬 Previsão de Estrutura Secundária de Proteínas

Um projeto de Deep Learning para prever a estrutura secundária de proteínas direto de dados cristalofrágicos. Envolve o tratamento de dados brutos (arquivos PDB/CIF) e o desenvolvimento da rede neural.

🔗 Repositório: Protein Secondary Structure
🎥 Resumidor de Vídeos com IA

Uma aplicação que utiliza LLMs (Grandes Modelos de Linguagem) para resumir vídeos do YouTube. O projeto usa técnicas de embeddings e sistemas de busca (retrieval) para extrair os pontos principais dos vídeos.

🔗 Repositório: Video Summarizer
💱 Pipeline de Dados de Câmbio Monetário

Construí uma pipeline de dados automatizada que consome dados de taxas de câmbio da FreeCurrencyAPI, processa as informações, empacota tudo com Docker e armazena em um Data Lake no MinIO. Um projeto focado em boas práticas de Engenharia de Dados e fluxos de ingestão reproduzíveis.

🔗 Repositório: Currency Exchange Data Pipeline
